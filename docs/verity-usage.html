Verity — how to use it

Prompt to production, proven.

Verity runs inside an AI coding assistant (Claude Code or OpenCode). You drive it with short commands. This guide shows the exact commands to type, in the right order, for real tasks — and the expert habits that make it sing.

🗣️ Commands YOU type (to the AI)

Things like /verity:vision. You type these into your AI assistant's chat. Each starts a specialist role.

⚙️ Commands the AI runs (for you)

Things like verity stage new. The AI runs these tool commands behind the scenes — you rarely type them yourself.
